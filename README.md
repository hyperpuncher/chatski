## tldr

- **sveltekit** + **electron**
- **keyboard driven**
- **no bloat**

## shortcuts

| shortcut       | action          |
| -------------- | --------------- |
| `ctrl` + `o`   | new chat        |
| `ctrl` + `m`   | toggle models   |
| `ctrl` + `b`   | toggle sidebar  |
| `ctrl` + `,`   | toggle settings |
| `ctrl` + `1-9` | switch chats    |

## dev

```bash
git clone https://github.com/hyperpuncher/chatski.git
cd chatski
bun install
bun run dev
```
